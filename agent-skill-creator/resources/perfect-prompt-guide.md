# 如何向 Agent 下达「创建 Skill」的完美提示词

本指南帮助你学会如何用一段精准的提示词，让 Agent 一次性输出高质量的完整 Skill。

---

## 核心原则：给 Agent 的信息越具体，输出的 Skill 越完美

Agent 创建 Skill 的质量直接取决于你提供的 **6 个维度** 的清晰度：

| 维度 | 你需要回答的问题 | 模糊 ❌ | 清晰 ✅ |
|------|----------------|---------|---------|
| **领域** | 这个 Skill 属于什么行业/专业？ | "帮我做营销" | "TikTok 美国市场 UGC 短视频营销" |
| **角色** | Agent 应该扮演谁？ | "一个助手" | "一位精通消费心理学的 TikTok 脚本策略师" |
| **核心任务** | 最终要完成什么？ | "写一些文案" | "将产品卖点转化为 30 秒 TikTok 脚本，含 Hook + 正文 + CTA" |
| **输入** | 用户提供什么数据？ | "产品信息" | "产品名称、3 个核心卖点、目标人群画像、竞品链接" |
| **输出** | 最终交付什么？ | "一段文字" | "2 套完整脚本（含时间戳、配音文案、画面分镜、标签矩阵）" |
| **质量标准** | 怎么算做得好？ | "写得好看" | "完播率 > 60%，俚语密度 ≥ 2 个/脚本，无 AI 机械词汇" |

---

## 提示词模板：一键生成完美 Skill

复制以下模板，替换 `【】` 中的内容，直接发给 Agent：

```
请使用 agent-skill-creator 技能，帮我创建一个完整的 Agent Skill。

## 技能概念
【用 1-2 句话描述这个 Skill 要做什么。例：将 TikTok 爆款脚本中的分镜画面描述，逐镜头转化为 AI 视频生成模型的结构化提示词。】

## 目标用户
【谁会使用这个 Skill？例：需要批量制作 TikTok 短视频的跨境电商团队。】

## 输入来源
【Skill 的数据从哪来？例：来自"tiktok爆款脚本制作skill"的输出，格式为包含时间戳和画面描述的 Markdown 脚本。】

## 期望输出
【最终交付物长什么样？例：每个分镜生成一段 100-150 词的英文视频提示词，包含镜头语言、主体、动作、光影和音频指令。】

## 多策略/多平台需求（如有）
【是否需要适配多个引擎/平台/框架？例：需要支持 Google Veo 3.1、OpenAI Sora 2、ByteDance Seedance 2.0 三个视频引擎。】

## 参考案例（如有）
【是否有现成的好案例可以参考？例：已有一个"油炸苹果"的成功脚本案例可作为 Few-Shot 样本。】

## 上下游链路（如有）
【这个 Skill 是否与其他 Skill 配合使用？例：上游接"tiktok爆款脚本制作skill"的输出，下游对接视频生成 API。】
```

## 实战示例：从提示词到完美 Skill

### 示例 1：最简版（适合快速原型）

```
请用 agent-skill-creator 技能帮我创建一个 Skill：
将用户输入的产品信息，自动生成 Amazon 商品标题和 5 条 Bullet Points。
需支持美国、日本、德国三个站点的本土化风格。
```

> 💡 即使是最简版，只要说清了 **核心任务 + 多策略需求**，Agent 就能生成一个合格的 Skill 骨架。

### 示例 2：详细版（适合生产级 Skill）

```
请使用 agent-skill-creator 技能，帮我创建一个完整的 Agent Skill。

## 技能概念
将客户的简单产品描述，转化为符合 TikTok 美国市场 UGC 风格的爆款短视频脚本。
输出包含标题、2 套完整脚本（含时间戳配音和画面分镜）、精准标签策略。

## 目标用户
跨境电商团队的内容运营人员，英语水平有限，需要 AI 直接输出可拍摄级别的脚本。

## 输入来源
用户手动提供：产品名称、核心卖点（1-3 个）、目标价位、产品图片链接。
还需要一份结构化的产品信息文件（按模板填写痛点、差异化优势、目标人群）。

## 期望输出
- 模块一：3 个带心理学触发器标注的爆款标题
- 模块二：2 套完整脚本，每套含 5 个阶段（Hook/Agitate/Solve/Payoff/CTA），每阶段有配音文案 + 画面描述 + 时间戳
- 模块三：10 个分层标签（品牌词 + 大流量 + 中流量 + 长尾）
- 全部英文输出，附中文翻译

## 多策略需求
需要支持 4 种已验证的爆款脚本框架：
A) 痛点反转型、B) 诱饵调包型、C) 极限视觉验证型、D) 多场景轰炸型。
Agent 需要根据产品属性自动选择最佳框架。

## 参考案例
已有 4 个成功案例可作为 Few-Shot：
1. 滚筒磨刀器（痛点反转型）
2. 油炸苹果（诱饵调包型）
3. 129N 爆款混剪（极限视觉验证型）
4. 胶带（多场景轰炸型）

## 上下游链路
下游对接"AI视频提示词生成skill"，需要确保输出的画面分镜描述可以被直接解析为视频提示词。

## 知识库需求
需要创建以下 resources 文件：
- 营销心理学触发器参考
- Hook 黄金三秒钩子指南
- CTA 行动号召模型
- TikTok 完播率最大化战略指南
- 产品信息结构化模板
```

> 💡 详细版提供了完整的 6 维信息 + 参考案例 + 上下游链路，Agent 可以一次性生成一个生产级别的高质量 Skill。

## 常见错误提示词 vs 改进方案

| # | ❌ 错误提示词 | 问题 | ✅ 改进后 |
|---|-------------|------|----------|
| 1 | "帮我做一个写文案的 Skill" | 无领域、无输出格式、无质量标准 | "帮我做一个为 Amazon 商品生成五点描述的 Skill，需支持美/日/德三个站点风格" |
| 2 | "创建一个 AI 视频 Skill" | 核心任务模糊 | "创建一个将 TikTok 脚本的画面描述转化为 Veo 3.1 视频提示词的 Skill" |
| 3 | "做一个营销 Skill，要很厉害" | "很厉害"不可量化 | "做一个 TikTok 脚本 Skill，目标完播率 > 60%，每脚本含 ≥ 2 个 Gen Z 俚语" |
| 4 | "参考我之前的 Skill 做一个新的" | 未指明参考哪个、参考什么 | "参考'tiktok爆款脚本制作skill'的框架选择机制，创建一个适用于 YouTube Shorts 的脚本 Skill" |

## 进阶技巧

### 1. 渐进式创建
如果你对 Skill 设计还没有完整想法，可以分步来：
1. 先给一个最简提示词，让 Agent 生成骨架
2. 审阅骨架后，追加细节："在 Step 2 中增加 XXX 规则"
3. 反复迭代直到满意

### 2. 借鉴已有 Skill
在提示词中直接引用已有 Skill 的结构：
> "参考 `AI视频提示词生成skill` 的质量自检清单格式，为新 Skill 也创建一份类似的清单"

### 3. 明确说明上下游
如果你的 Skill 是工作流链条中的一环，必须说清楚：
- 上游 Skill 输出什么格式的数据给我？
- 我的 Skill 输出什么格式的数据给下游？
- 这样 Agent 才能确保接口兼容
