# AI 视频提示词生成 Skill

将 TikTok 爆款脚本中的分镜画面，转成可直接用于 Seedance 2.0、Veo 3.1、Sora 2 的结构化视频提示词。

---

## 这是什么

这是一个面向 TikTok UGC 视频制作的 Agent Skill。它接收脚本里的“画面”描述，输出：

- 合并后的镜头段落
- 适配目标引擎的 Prompt
- 明确的 POV / 真人出镜规则
- 需要准备的参考素材清单

核心目标：

- 画面还原度高
- UGC 原生感强
- 角色、空间、道具一致
- 尽量减少片段数量

---

## 核心能力

| 能力 | 说明 |
|---|---|
| 脚本转视频 Prompt | 从脚本画面描述中提取场景、动作、结果、音效并转成 Prompt |
| 三大引擎适配 | 支持 Seedance 2.0、Sora 2、Veo 3.1 |
| POV 模式 | 支持第一人称 POV、胸前固定运动相机视角、双手操作逻辑 |
| 少量片段原则 | 自动将脚本尽量合并为 2-4 段 |
| UGC 风格控制 | 压制影棚感和广告感，强化真实手机实拍感 |

---

## Seedance 2.0 新规则

### 1. 引用语法改成 bare `@`

Seedance 2.0 Prompt 正文不再使用旧的图片/视频编号式标签。

正文里只保留 bare `@` 作为挂参考的示意，可以直接贴在被引用对象前面。

正确示例：

```text
视频场景，第一人称 POV 胸前固定运动相机视角，偏白男性手，黑色运动表，灰色连帽衫袖口。
在@场景中，画面更亮更干净，像真实 home cook 顺手拍下来的成果展示。
左手按住砧板上的番茄，右手用磨好的@刀按照@的切削动作和构图切出薄片。
最后双手把 @电动磨刀器和刚切好的食材一起摆在台面中央。
```

补充约束：

- 只要是切番茄、洋葱、肉类、水果、面包等食材，正文必须明确写成“在砧板上切”
- 不要写成食材直接放在台面上被切，也不要写成食材悬空被切

### 2. 钝刀镜头必须补视频参考

只要脚本里出现这些失败切割画面，就必须准备独立的失败动作视频：

- 番茄压扁
- 洋葱卡顿
- 鸡肉拖切
- 面包发涩
- 牛排压变形
- 纸张扯裂
- 绳子切不断

正文里同样只保留 bare `@`：

```text
左手按住砧板上的番茄，右手用@刀按照@的切削动作和构图下压，把番茄切烂出汁。
```

### 3. 图参和视频参分工

- 图参负责锁定对象和场景，不重复写图里已经清楚可见的材质、身份或"同一把""同一处"。
- 视频参只负责动作、运镜、节奏、构图，不要写成"@的钝刀失败节奏"这种语义过重的标签。

推荐改写：

```text
错误：@同一美式郊区厨房的木质岛台前
正确：在@场景中

错误：同一把 @钝口主厨刀
正确：用@刀

错误：按照@的钝刀失败节奏撕裂 A4 纸和压塌番茄
正确：按照@的切削动作和构图切西红柿，把西红柿切烂；按照@的切削动作和构图切 A4 纸，把纸切烂
```

### 4. 时间线按现实时长切分

- 时间线不是按“文案分量”分时长，而是按真实动作耗时分时长。
- 像“拨回配件”“放下箱子”“拿起刀”这类动作通常只需要 0.5 到 1.5 秒。
- 像“连续切一刀失败”“连续切一刀成功”“开箱后扫过内部结构”这类动作通常 1.5 到 3 秒。
- 只有连续打磨、持续展示、连续切多片这类过程，才适合 3 到 5 秒。
- 必要时允许使用 0.5 秒粒度，例如 `[1.0s-1.5s]`。

推荐改写：

```text
错误：[0s-4s] 她把几件廉价小拉刀器和零散配件快速拨回半开的抽屉里。
正确：[0s-1.0s] 她把几件廉价小拉刀器和零散配件快速拨回半开的抽屉里。[1.0s-1.5s] 她顺手把抽屉推回。

错误：[4s-7s] @黑色 SHARPAL 202H 硬壳箱被有分量地放到桌面中央。
正确：[1.5s-2.0s] @黑色 SHARPAL 202H 硬壳箱被有分量地放到桌面中央。[2.0s-3.0s] 镜头顺势压近箱体正面和边角。
```

### 5. 缺少失败视频时不要硬出 Prompt

如果脚本包含钝刀失败镜头，但没有对应失败动作视频：

- 先提示补素材
- 不输出最终版 Seedance Prompt
- 不用纯文本降级
- 不擅自复用磨刀视频或锋利切割视频

## 快速开始

### 输入来源

推荐直接使用 `tiktok爆款脚本制作skill` 的模块二输出。

### 最小输入要求

脚本每个阶段都要有清晰的 `画面` 描述，至少能看出：

- 场景
- 主体
- 道具
- 动作
- 结果

### 使用示例

```text
请将以下 TikTok 脚本转化为 Seedance 2.0 视频提示词。

要求：
1. Prompt 正文只使用 bare @
2. 不使用旧的图片/视频编号式标签
3. 图参只锁对象/场景，视频参只锁动作/运镜/节奏/构图
4. 时间线必须贴合真实动作耗时，必要时使用 0.5 秒粒度
5. 钝刀失败镜头必须补失败动作视频
6. 若缺少该视频，先提示我补素材，不要直接输出最终 Prompt

视角：POV

[粘贴脚本内容]
```

## 5 段式公式

每个合并后片段都按下面的思路生成：

```text
[镜头语言] + [主体] + [动作序列] + [环境/光影] + [风格/音频]
```

## 支持的引擎

### Seedance 2.0

- 中文输出
- 正文只用 bare `@`
- 用素材锁动作和外观
- 钝刀失败镜头必须补失败动作视频
- 音频直接写在正文末尾，不使用单独的音频引用标签

### Sora 2

- 英文自然叙事
- 强调流体、碰撞、重力等物理过程
- 适合戏剧化镜头

### Veo 3.1

- 英文 5 段式 Prompt
- 原生 9:16 + 音频能力强
- 末尾补 `Audio:` 或 `SFX:`

## 质量检查重点

每次输出前至少检查：

1. 是否尽量合并成 2-4 段
2. 是否保留真实 UGC 质感
3. Seedance 正文里是否完全清除旧的图片/视频编号语法
4. 钝刀失败镜头是否真的准备了失败动作视频
5. POV 下是否写清楚双手分工
6. 是否避免广告感和危险动作
