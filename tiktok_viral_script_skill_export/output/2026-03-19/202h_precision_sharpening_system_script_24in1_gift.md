  **画面建议**: 明确还是同一把主厨刀。先闪回它撕裂 A4 纸、压扁番茄的失败画面，再展示完整流程后顺滑切开悬空 A4 纸，接着干净切出番茄薄片。最后给刃口一个冷光微距，让镜面反光说明“不是礼盒摆拍，是真的磨出来了”。
