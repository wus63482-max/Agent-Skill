**Hook 类型**: Visual Proof（视觉验证）+ Fail vs Success（对比评测）  
**心理触发器**: 专业人群圈定 -> 旧方案一失败 -> 旧方案二失败 -> 202H 真实控角解决 -> 同刀验证 + 专业 CTA

- **[0-8s] Hook** [~9 words]:
  **口播**: Freehand stone, same knife, steady hands. Still tearing paper.
  **【加重语气配字幕】**: same knife, still tearing
  **屏显文字 (红/黄大字)**: SAME KNIFE. STILL FAILING.
  **画面建议**: 前 2 秒直接给同一把主厨刀切对折 A4 纸失败，纸边被拖拽、撕裂。镜头马上切到手工水磨石场景，手在尽量稳住角度来回磨，动作认真但结果还是不行，第一秒就把“专业用户最烦的失败感”砸出来。
  [中文翻译]: 手磨石，同一把刀，手已经尽量稳了，结果还是在撕纸。

- **[8-15s] Agitate** [~12 words]:
  **口播**: Okay, cheap pull-through next. Same edge, full pressure. Tomato still gets crushed.
  **【加重语气配字幕】**: cheap pull-through, still gets crushed
  **屏显文字**: TRY AGAIN. SAME PROBLEM.
  **画面建议**: 换到便宜拉刀器或散装小磨刀工具，还是同一把刀。镜头给出“全力再试一次”的真实感，然后切到熟番茄被压扁而不是切开的近景，强调“换个便宜方案也没救”。
  [中文翻译]: 好，那再换便宜拉刀器试试。同一个刀口，用力拉，番茄还是照样被压扁。

- **[15-26s] Solve** [~20 words]:
  **口播**: Now switch to the 202H. Lock the angle, clamp the blade, run all seven plates. That's real control, for real.
  **【加重语气配字幕】**: lock the angle, seven plates, real control
  **屏显文字**: LOCK IT. DIAL IT. SHARPEN IT.
  **画面建议**: 镜头快速切到 202H 主体。依次特写调角杆、角度刻度、金属夹具夹稳刀身、安装磨板、导杆顺滑推进。磨板表面只滴少量润滑液，不出现火花，不整机泡水，重点放在“精准控制感”和“完整 progression”。
  [中文翻译]: 现在换 202H。锁定角度，夹稳刀身，7 片磨板完整走一遍。这才是真正的可控，真的。

- **[26-33s] Payoff** [~10 words]:
  **口播**: Same knife. Clean A4 slice, clean tomato slice, mirror edge.
  **【加重语气配字幕】**: clean A4, mirror edge
  **屏显文字**: SAME KNIFE. DIFFERENT RESULT.
  **画面建议**: 不闪回失败镜头，直接给成功验证。还是同一把刀，顺滑切开对折 A4 纸，再干净切出番茄薄片，最后给刃线一个冷光反射特写，突出“结果不是说出来的，是磨出来的”。
  [中文翻译]: 同一把刀。A4 纸干净切开，番茄干净切开，刃口还有镜面反光。

- **[33-39s] CTA** [~11 words]:
  **口播**: If you actually care about your edge, this system hits different.
  **【加重语气配字幕】**: care about your edge, hits different
  **屏显文字**: FOR PEOPLE WHO ACTUALLY CARE
  **画面建议**: 镜头扫过打开后的 202H 套装内部，磨板、夹具、滴瓶整齐排布，最后停在系统主体和刀同框的位置。收尾语气保持专业，不报价格，不喊夸张促销。
  [中文翻译]: 如果你是真的在意刀口的人，这套系统的感觉就是不一样。
