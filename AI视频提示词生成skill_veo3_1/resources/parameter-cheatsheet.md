# Veo 3.1 参数速查表

## 引擎参数

| 参数 | 规格 |
|------|------|
| **分辨率** | 4K / 1080p |
| **画幅 (TikTok)** | ✅ 原生 9:16 |
| **单次时长** | 最高 8s |
| **延伸能力** | Scene Extension → 60s+ |
| **帧率** | 24fps |
| **原生音频** | ✅ 48kHz（对白+音效+配乐） |
| **角色一致性** | Ingredients（参考图） |
| **参考素材** | 图片参考 |

---

## TikTok 场景推荐配置

```
Resolution: 4K
Aspect Ratio: 9:16
Duration: 3-8s (per shot, max 8s)
Audio: Enabled (native)
```

## 分镜时长建议

| 叙事阶段 | 推荐时长 | 说明 |
|---|---|---|
| Hook | 3s | 短促有力，瞬间抓住眼球 |
| Bridge | 2-3s | 如有自然过渡段 |
| Agitate | 5s | 快速堆叠痛点画面 |
| Solve | 5s | 产品开箱到上手操作 |
| Payoff | 7-8s | 多场景效果蒙太奇（贴近 8s 上限） |
| CTA | 3-4s | 产品居中+购买指引 |

---

## Prompt 长度限制

> ⚠️ 每段提示词严格控制在 **100-200 个英文单词**（约 3-6 句话）。超出会导致引擎忽略部分指令。

---

## Veo 3.1 适用场景

| 你的需求 | 是否适用 |
|----------|----------|
| 全流程制作，需要口播同步 | ✅ 最适合 |
| 完整 9:16 竖屏流程 | ✅ 最适合 |
| 纯文字描述，无参考素材 | ✅ 最适合 |
| 需要原生音频 | ✅ 最适合 |
| 有大量物理交互 | ❌ 用 Sora 2 |
| 有参考素材 | ❌ 用 Seedance 2.0 |
