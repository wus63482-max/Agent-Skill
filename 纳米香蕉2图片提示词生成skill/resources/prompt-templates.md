# 使用模板 & 前置检查清单

## ⚠️ 使用前必读：前置检查

在使用本技能前，请逐条确认：

| # | 检查项 | 状态 | 说明 |
|---|---|---|---|
| 1 | **已明确生成场景？** | ☐ | 角色肖像 / 产品展示 / 6 宫格分镜 — 三选一或组合 |
| 2 | **角色/产品描述已准备？** | ☐ | 至少提供年龄、性别、族裔（角色）或产品名称、外观（产品） |
| 3 | **目标用途已确定？** | ☐ | 用于视频模型参考图？电商主图？分镜预审？ |
| 4 | **画面比例已选择？** | ☐ | 默认 3:4（角色）/ 9:16（TikTok）/ 1:1（产品）/ 3:2（分镜） |

> [!CAUTION]
> 若要用于 Seedance 2.0 视频参考，角色肖像**必须走完五步法**（包含洗人提示词），否则视频模型的角色一致性无法保障。

## 📋 Prompt 模板

### 模板 1：UGC 角色肖像全流程（推荐）

同时引用全部资源，一次性生成角色全套参考图 + 洗人提示词：

```
@SKILL.md
@resources/ugc-style-keywords.md
@resources/prompt-formula-cheatsheet.md

请为以下角色生成纳米香蕉 2 的完整五步法图片提示词：

角色描述：
- 年龄：28 岁
- 性别：女性
- 族裔：白人
- 风格：美区 TikTok 邻家感
- 穿搭：日常居家休闲
- 用途：Seedance 2.0 视频参考

请生成：
1. 角色小传（中英双语）
2. 5 组多角度参考图提示词（正面/侧面/全身/动作/情绪）
3. 洗人提示词（中文版 + 英文版）
```

## 🔔 常见问题提醒

| 问题 | 原因 | 解决方案 |
|---|---|---|
| 生成的图片太"精致" | 未引用 `ugc-style-keywords.md` | 添加 @引用该文件，加入 UGC 关键词 |
| 角色跨图不一致 | Character Weight 太低 | 设置 Character Weight 为 95-100 |
| 皮肤像"蜡像"感 | 缺少皮肤瑕疵描述 | 添加 freckles、visible pores、uneven skin tone |
| 图片过于对称 | 构图太专业 | 加入 randomly framed、slightly off-center |
| 文字渲染失败 | 未用双引号 | 文字内容必须用英文双引号包裹 |
| 提示词被截断 | 超出词数上限 | 控制在 60-120 词以内 |
| 洗人提示词无效 | 描述不够具体 | 必须包含5官+肤色+瑕疵+发型+穿搭全维度 |
| 6 宫格角色串脸 | 缺少一致性锚定 | 每条提示词重复角色核心特征 |
