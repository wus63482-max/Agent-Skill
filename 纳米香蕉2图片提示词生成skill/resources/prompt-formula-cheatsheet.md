# 提示词公式与参数速查 — 纳米香蕉 2

## 核心提示词公式

### 5 段式公式（通用）

```
[主体 Subject] + [动作/姿态 Action/Pose] + [环境/场景 Environment] + [光影/风格 Lighting/Style] + [画质/参数 Quality/Parameters]
```

### 各场景公式变体

| 场景 | 公式 | 词数限制 |
|------|------|---------|
| **角色肖像** | [五官+肤色+瑕疵] + [表情+姿态] + [生活化场景] + [自然光影] + [手机画质+3:4] | 60-100 词 |
| **产品特写** | [产品+材质+细节] + [摆放状态] + [使用环境] + [光影] + [画质+1:1] | 40-80 词 |
| **UGC 场景产品** | [产品+材质] + [在生活台面] + [周边杂物] + [室内灯光] + [手机画质+4:3] | 60-100 词 |
| **POV 使用图** | [手部描述+产品] + [使用动作] + [场景] + [自然光影] + [手机+9:16+POV] | 60-100 词 |
| **6 宫格分镜** | [6-panel layout+角色描述] + [每格画面] + [一致性指令] + [草稿风格] + [3:2] | 80-150 词 |

## 洗人提示词公式

### 真人出镜版（完整人物）

```
[年龄+族裔+性别] + [脸型+五官核心特征] + [发型发色] + [皮肤瑕疵/不完美细节] + [穿搭] + [一个生活化微动作]
```

### POV 版（仅手部）

```
[一双手+肤色+性别] + [手部特征：指甲/疤痕/配饰] + [手腕配饰] + [袖口/衣物细节] + [手机型号画质]
```

## 角色一致性参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **Character Weight** | 90-100 | 角色一致性权重，高值 = 跨图不"串脸" |
| **Dynamic Reasoning** | High / Dynamic | 处理复杂空间关系和多角色场景 |
| **Style** | Photographic | 写实风格输出 |

## 多图参考速查

### 公式

```
[主语描述] + [明确的图像索引关联] + [具体的动作/环境] + [画质/画幅参数]
```

### 三大用法

| 用法 | 上传内容 | 提示词关键写法 |
|------|---------|---------------|
| **角色+风格融合** | 图1(角色) + 图2(风格) | `保留图1中人物的长相特征，将其置于图2的场景中` |
| **姿势+材质参考** | 图1(姿势) + 图2(产品) | `参考图1的姿势，手中拿着图2中的产品` |
| **角色一致性(三视图)** | 图1+图2+图3(同一角色) | `根据图1、图2、图3中人物的统一长相，确保一致` |

### 限制与注意

| 项目 | 限制 |
|------|------|
| 推荐参考图数量 | **2-4 张**（超过 4 张易杂乱） |
| 指代方式 | 必须用"图1"/"图2"或"第一张图"/"第二张图" |
| 负面修正 | 不支持 Negative Prompt，用描述规避：`不要有图1中的杂乱线条` |
| 中文支持 | ✅ 可全程中文，理解力极强 |

## 提示词长度控制

| 场景 | 推荐词数 | 最大词数 | 说明 |
|------|---------|---------|------|
| 角色肖像 | 60-80 词 | 100 词 | 核心特征优先 |
| 产品特写 | 40-60 词 | 80 词 | 精简聚焦产品 |
| 场景化产品 | 60-80 词 | 100 词 | 环境细节适度 |
| POV 使用图 | 60-80 词 | 100 词 | 动作描述为重 |
| 6 宫格分镜 | 80-120 词 | 150 词 | 含 6 格描述 |

> ⚠️ 超出模型处理上限（约 150 词）会导致后半段指令被忽略。宁可精简，不要冗长。

## 局部重绘 (Inpainting) 速查

### 常见修正场景

| 问题 | 涂抹区域 | Inpainting 提示词 |

| 光影太专业 | 全图 | `harsh phone flash, slight overexposure, shadows` |
| 五官太精致 | 面部 | `add natural asymmetry, slightly uneven eyebrows` |
| 妆容太完美 | 面部 | `bare face, no makeup, natural lip color` |

## 文字渲染技巧

| 需求 | 提示词写法 | 说明 |
|------|-----------|------|
| 标牌文字 | `A neon sign that says "OPEN"` | 必须用双引号 |
| 产品标签 | `Label clearly reading "Organic"` | 明确说明位置 |
| T恤文字 | `T-shirt with text "YOLO" printed on front` | 指定衣物位置 |
| 手写文字 | `A handwritten note saying "Call me"` | 声明手写风格 |
