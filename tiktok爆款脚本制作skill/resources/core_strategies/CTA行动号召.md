# CTA (Call-to-Action) 通用行动号召框架

> 本文件提供跨品类通用的 CTA 设计框架。Agent 应根据产品价位、用户决策阶段和心理触发器选择最合适的 CTA 组合。

---

## 一、CTA 选择决策树

```
产品价位判断
├── 低价 ($5-$25)  → 优先: 冲动型 CTA (紧迫感 + 低门槛)
│   例: "Only $15, link in bio before it sells out"
├── 中价 ($25-$80) → 优先: 价值型 CTA (价值锚定 + 社会认同)
│   例: "Best $40 I ever spent, 50,000 people agree"
└── 高价 ($80+)    → 优先: 信任型 CTA (风险消除 + 权威背书)
    例: "30-day money back, I tested it for 6 months"
```

---

## 二、按心理学触发器分类的 CTA 模板

### 1. 稀缺 & 紧迫感 (Scarcity & Urgency)
**心理原理**: 物品稀缺时感知价值飙升，害怕错过驱动立即行动

| CTA 模板 (英文) | 适用品类 | 使用语境 |
|---|---|---|
| "They're running a sale RIGHT NOW, link's in my bio" | 通用 | 限时促销 |
| "Only X left in stock, don't sleep on this" | 通用 | 库存紧张 |
| "This deal ends tonight, I'm literally buying 3 more" | 通用 | 限时折扣 |
| "I found this before it went viral, get it while you can" | 新品/小众 | 早期发现 |
| "Last time I posted this it sold out in 2 hours" | 已验证爆品 | 复购号召 |

### 2. 价值锚定 (Value Anchoring)
**心理原理**: 用对比价格或替代方案成本锚定产品的"超值感"

| CTA 模板 (英文) | 适用品类 | 使用语境 |
|---|---|---|
| "Best $X I've ever spent, not even close" | 通用 | 性价比突出 |
| "This replaced my $XXX [expensive alternative]" | 替代品 | 替代高价方案 |
| "For less than a coffee a day you get [benefit]" | 订阅/低价品 | 日均成本拆分 |
| "I was gonna spend $XXX on [alternative], then I found this" | 通用 | 省钱叙事 |
| "Literally cheaper than [relatable daily expense]" | 低价品 | 价格对比 |

### 3. 社会认同 (Social Proof)
**心理原理**: 不确定时人们会参考他人行为

| CTA 模板 (英文) | 适用品类 | 使用语境 |
|---|---|---|
| "X people bought this from my last video" | 已验证爆品 | 复购/追加 |
| "My whole friend group has one now" | 社交属性强 | 传播效应 |
| "There's a reason this has X reviews on Amazon" | Amazon 爆品 | 评价背书 |
| "POV: you're the last person to find out about this" | 通用 | FOMO + 社会压力 |
| "OK but why is everyone buying this rn" | 趋势品 | 跟风效应 |

### 4. 风险消除 (Risk Reversal)
**心理原理**: 消除购买犹豫，降低感知风险

| CTA 模板 (英文) | 适用品类 | 使用语境 |
|---|---|---|
| "Try it risk-free, they have a X-day return policy" | 高价品 | 犹豫用户 |
| "I was skeptical too but honestly just try it" | 通用 | 真实体验 |
| "If you hate it just return it, but you won't" | 通用 | 自信推荐 |
| "Free shipping and free returns, literally zero risk" | 通用 | 政策利好 |

### 5. 好奇心驱动 (Curiosity-Driven)
**心理原理**: 制造信息缺口，驱动探索行为

| CTA 模板 (英文) | 适用品类 | 使用语境 |
|---|---|---|
| "Go check the link and thank me later" | 通用 | 轻松号召 |
| "You NEED to see the reviews on this thing" | 通用 | 引导浏览 |
| "Wait till you see what it does to [specific use case]" | 效果类产品 | 预留悬念 |

### 6. 痛点解决 (Problem Resolution)
**心理原理**: 损失厌恶——停止"正在亏损"的状态

| CTA 模板 (英文) | 适用品类 | 使用语境 |
|---|---|---|
| "Stop wasting money on [inferior solution]" | 替代品 | 替代方案差 |
| "If you're still dealing with [pain point], just get this" | 通用 | 直接解决 |
| "Your [problem] ends today, link in bio" | 通用 | 终结痛苦 |
| "I wish I found this sooner, would've saved me [time/money]" | 通用 | 后悔叙事 |

## 三、CTA 视觉配合建议

| 配合动作 | 说明 |
|---|---|
| **手指指向** | 在说 CTA 时手指向屏幕左下角（TikTok 购物车位置） |
| **文字弹出** | 用画面文字强化关键词（价格/折扣码），用红色或黄色高亮 |
| **声音变化** | CTA 部分语速略慢、语气更诚恳，避免"喊麦"感 |
| **二次钩子** | CTA 前加一句"wait, one more thing..."防止提前划走 |
