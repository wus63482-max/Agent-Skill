# AI 视频生成引擎参数速查表 — TikTok 竖屏场景

## 关键参数对比

| 参数 | Veo 3.1 (Google) | Sora 2 Pro (OpenAI) | Seedance 2.0 (ByteDance) |
|---|---|---|---|
| **分辨率** | 4K / 1080p | 1080p / 1792×1024 | 4K |
| **画幅 (TikTok)** | ✅ 原生 9:16 | ✅ 9:16 | ✅ 9:16 |
| **单次时长** | 最高 8s | 4s / 8s / 12s / 25s (Pro) | 可配置 |
| **延伸能力** | Scene Extension → 60s+ | Timeline Editor 多段 | 叙事规划器多镜头 |
| **帧率** | 24fps | 24fps / 30fps | 可配置 |
| **原生音频** | ✅ 48kHz (对白+音效+配乐) | ✅ (对白+音效+配乐) | ✅ (音画对齐/卡点) |
| **角色一致性** | Ingredients (参考图) | @Cameo (锁定角色) | @Image (身份锁定) |
| **参考素材** | 图片参考 | 图片参考 | 图/视/音/文 四模态 |

## TikTok 场景推荐配置

### 通用推荐（适用大多数场景）

```
Resolution: 4K
Aspect Ratio: 9:16
Duration: 3-8s (per shot)
Audio: Enabled (native)
```

### 分镜时长建议

| 叙事阶段 | 推荐时长 | 说明 |
|---|---|---|
| Hook | 3s | 短促有力，瞬间抓住眼球 |
| Bridge | 2-3s | 如有自然过渡段 |
| Agitate | 5s | 快速堆叠痛点画面 |
| Solve | 5s | 产品开箱到上手操作 |
| Payoff | 7-8s | 多场景效果蒙太奇 |
| CTA | 3-4s | 产品居中+购买指引 |

## 引擎选择建议

| 你的需求 | 推荐引擎 | 原因 |
|---|---|---|
| 全流程制作，需要口播同步 | **Veo 3.1** | 原生对白同步、4K 画质 |
| 有大量物理交互（飞溅、火花、碰撞） | **Sora 2** | 物理模拟是其核心优势 |
| 有参考素材（产品图、动作视频） | **Seedance 2.0** | 四模态锚定精准复刻 |
| 需要音乐卡点 | **Seedance 2.0** | Rhythm Sync 自动对齐 |
| 纯文字描述，无参考素材 | **Veo 3.1** | 5 段式公式理解最精准 |

---

## Prompt 长度限制

> ⚠️ 每段提示词严格控制在 **100-150 个英文单词**（约 3-6 句话）。超出会导致引擎忽略部分指令。
