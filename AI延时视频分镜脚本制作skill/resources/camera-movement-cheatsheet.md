# 运镜技巧速查表

> 本文件提供延时摄影中常用运镜方式的速查参考，帮助 Agent 为每个分镜选择最合适的运镜方式。

---

## 运镜方式对比

| 运镜方式 | 视觉效果 | 最佳适用场景 | 情绪表达 | 搭配建议 |
|---------|---------|-------------|---------|---------|
| **固定机位延时** | 画面静止、时间加速 | 建筑搭建全过程、交通流变化 | 客观、纪录、震撼 | 通常作为主镜头使用，占比50%+ |
| **缓慢滑轨横移** | 平滑扫描、逐步揭示 | 外墙细节展示、天际线全景扫描 | 叙事感、仪式感、期待 | 适合中段过渡镜头，展示细节变化 |
| **缓慢推进/拉远** | 靠近或远离主体 | 从全景到细节/从细节到全景 | 聚焦→发散 或 发散→聚焦 | 适合开场或结尾的引导性镜头 |
| **无人机直线上升** | 从地面到鸟瞰揭示 | 完工揭幕、全局展示 | 揭示感、成就感、震撼 | 最常用的收尾大招运镜 |
| **无人机弧线环绕** | 360°/270° 环绕主体 | 完工建筑展示、城市夜景全景 | 沉浸、立体、包裹感 | 比直线上升更有动态感 |
| **无人机俯冲下降** | 从高空俯冲至近景 | 开场建立空间感 | 快速进入、紧迫、冲击力 | 适合节奏快的开场 |
| **微距延时** | 超近距离拍摄缓慢变化 | 花朵绽放、冰雪融化、锈蚀 | 宁静、微观、奇迹感 | 自然生态类主运镜 |

## 运镜搭配原则

### 基础搭配规则
1. **全片至少使用 2 种以上运镜方式**，避免单调
2. **固定机位与运动机位交替使用**，形成动静节奏感
3. **收尾镜头必须是全片运动幅度最大的**（通常为无人机航拍）

### 按场景类型推荐搭配

#### 🏗️ 建筑搭建类
```
镜头1-3: 固定机位延时（同机位保持空间连续性）
镜头4:   滑轨横移（展示外观细节变化）
镜头5:   无人机上升揭示（收尾大招）
```

#### 🌃 城市景观类
```
镜头1: 固定高处俯瞰（建立城市全景）
镜头2: 滑轨平移扫描（黄昏天际线）
镜头3: 固定机位延时（华灯初上的转折时刻）
镜头4: 无人机弧线环绕（深夜全景揭示）
```

#### 🌿 自然生态类
```
镜头1: 微距延时（种子萌芽/花苞初现）
镜头2: 缓慢推进（生长过程细节）
镜头3: 固定机位延时（绽放高潮）
镜头4: 缓慢拉远至全景（揭示全貌）
```

> [!TIP]
> AI 视频引擎（Seedance/Veo 等）对固定机位和缓慢运动的支持最好。无人机环绕等复杂运镜在 Prompt 中需要更精确的描述词。
